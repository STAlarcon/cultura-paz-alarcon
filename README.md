# Cultura de Paz — Prevención del Ciberacoso

Este sitio promueve una **cultura de paz digital**, ayudando a prevenir el **ciberacoso** y a fomentar el respeto en redes sociales. Contiene señales de alerta, acciones de autocuidado y contactos de apoyo.

**Tema:** Violencia digital y ciberacoso  
**Fuentes:**

- [UNICEF México](https://www.unicef.org/mexico/)
